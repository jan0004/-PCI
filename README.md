# PCI Pavement Condition Index
# excel VBA 自定義模組
--------------------------
excel 模組化 PCI折減值計算/n
道路評分表中值減值需要查詢許多表格來求出折減係數/n
利用此模組可以塊數建立折減值/n
---------------------------
根據「市區道路管理維護與技術規範手冊研究-柔性撲面損壞調查手冊」-內政部營建署-民國91年四月版/n
@ 折減係數換算函數-roadDV/n
--函數使用方法--/n
roadDV(損害類型,密度)/n
備註:損害類型為字串，如1M，即 龜裂中度/n
    密度為(%)，四捨五入至小數點後一位/n
@ 修正折減係數換算函數-CDV/n
--函數使用方法--/n
CDV(q值,total)/n
備註:q值為整數，如1,2,3...，即 大於2之數值/n
    total 為浮點數，為折減值之總和/n
